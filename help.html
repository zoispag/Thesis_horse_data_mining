Επιλέξτε τη διοργάνωση για την οποία επιθυμείτε να συλλέξετε κούρσες και να δημιουργήσετε αρχείο .idml (InDesign)

Για κάθε μία από τις διοργανώσεις, Αγγλίας, Γαλλίας ή Ν. Αφρικής μπορείτε να πατήσε το εκτέλεση προκειμένου να ξεκινήσει η συλλογή.

Επίσης, για κάθε μία από τις διοργανώσεις υπάρχει κουμπί Μεταφόρτωση, όπου μπορείτε να μεταφορτώσετε αρχεία απαραίτητα για την επεξεργασία των κουρσών.

Τέλος, για της κούρσες Γαλλίας υπάρχουν οι Εκτιμήσεις, οι οποίες παράγουν τις κορυφαίες εκτιμήσεις για κάθε άλογο.
